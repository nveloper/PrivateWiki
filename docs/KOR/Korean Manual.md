# Private Wiki 운영/이용 메뉴얼

본 문서는 Node.js 백엔드 서버를 기반으로 구동되는 **Private Wiki** 의 운영 및 사용법을 설명합니다. 데이터베이스 없이 마크다운(`.md`) 파일들만으로 동작하며, 웹 UI에서 직접 문서의 수정/생성/삭제가 가능한 버전입니다.

---

## 1. 서버 실행 및 초기 설정

### 1.1 서버 실행 방법
1. 터미널을 열고 프로젝트 최상위 폴더(`/`)로 이동합니다.
2. `npm install`을 입력해 필요한 패키지를 설치합니다.
3. `node server.js` 명령어를 실행하여 서버를 켭니다.
4. 브라우저에서 `http://localhost:8180` 로 접속합니다.

### 1.2 관리자 로그인 (편집 권한)
문서를 수정, 삭제하거나 새 문서를 만들려면 우측 상단의 **자물쇠(Login)** 아이콘을 눌러 로그인해야 합니다.
- **초기 비밀번호:** `admin`
- 비밀번호 변경 방법: 서버가 한 번 실행되면 프로젝트 루트에 `auth.json` 파일이 생성됩니다. 이 파일 안의 비밀번호를 직접 수정하고 서버를 재시작하면 변경됩니다.

## 2. 기본 사용법 (위키 이용하기)

### 2.1 문서 열람 및 내비게이션
- **사이드바 (목록표):** 좌측의 트리 구조를 통해 폴더와 문서를 탐색할 수 있습니다.
- **문서 내부 목차 (TOC):** 마크다운 내에 `#` (제목) 태그를 사용하면 우측 하단에 목차 버튼이 생기며, 클릭 시 해당 단락으로 부드럽게 스크롤됩니다.
- **검색:** 상단의 돋보기 아이콘을 눌러 위키 내의 모든 문서 내용을 검색할 수 있습니다.
- **랜덤 문서:** 상단의 주사위 아이콘을 누르면 작성된 문서 중 무작위로 하나를 골라 보여줍니다.

### 2.2 문서 편집 및 관리 (관리자 전용)
로그인을 완료하면 다음과 같은 기능이 활성화됩니다.
- **문서 수정 (Edit):** 우측 하단의 연필 아이콘을 누르면 마크다운 편집기가 열립니다.
- **항목 생성 (Create):** 파일 트리의 특정 폴더 옆에 나타나는 `+` 버튼을 눌러 새 폴더나 `.md` 문서를 생성할 수 있습니다.
- **이름 변경/삭제 (Rename/Delete):** 문서나 폴더 이름 옆의 톱니바퀴 아이콘을 누르면 이름을 바꾸거나 삭제할 수 있습니다.

### 2.3 이미지 업로드
편집 모드 하단의 **Image Upload** 버튼을 통해 이미지를 서버에 업로드할 수 있습니다.
- 업로드 완료 시 자동으로 `![이미지이름](/uploads/images/파일명.png)` 형태의 마크다운 코드가 생성되므로, 이를 복사해 문서에 붙여넣으면 됩니다.

## 3. UI 및 테마 설정

우측 상단의 **톱니바퀴(Settings)** 아이콘을 누르면 사이트 전체에 적용되는 설정을 변경할 수 있습니다. 
(이 설정은 서버의 `settings.json`에 저장되어 모든 접속자에게 반영됩니다. 단, 테마 모드와 컬러 팔레트는 개인 브라우저에만 저장됩니다.)

- **Site Title:** 브라우저 탭과 사이트 좌측 상단에 표시될 이름을 지정합니다.
- **Home Document:** 처음 접속했을 때 띄워줄 메인 화면 문서를 지정합니다. (예: `folder1/welcome.md`)
- **Theme Mode:** 
  - `System`: 운영체제의 다크 모드 설정을 따라갑니다.
  - `Light` / `Dark`: 항상 밝게 혹은 어둡게 고정합니다.
  - `Schedule`: 지정된 시간(예: 18:00 ~ 06:00)에만 자동으로 다크 모드로 전환됩니다.
- **Primary Color:** 사이트의 메인 색상(버튼, 선택된 폴더 등)을 변경합니다. 
- **Font & Favicon Upload:** 내 컴퓨터의 폰트나 파비콘 이미지를 업로드하고 즉시 적용할 수 있습니다.

## 4. 특수 매크로 기능

마크다운 파일 내에 특정한 텍스트 구문을 입력하면 동적인 기능이 작동합니다.

**1. 리다이렉트 매크로**
특정 문서 접속 시 다른 문서로 자동 이동시킵니다.
```markdown
{'{REDIRECT:폴더명/이동할문서.md}'}
```

**2. 디데이(D-Day) 카운트다운 매크로**
특정 날짜까지 남은(혹은 지난) 일수를 자동으로 계산해 줍니다.
```markdown
개막일까지 {{DAYS_UNTIL:2026-12-25}}일 남았습니다.
```
