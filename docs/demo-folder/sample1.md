# 한국어 설명서

이 문서는 데모용 마크다운 파일입니다.

## 1. 텍스트 강조 및 스타일
이 위키는 표준 **GFM (GitHub Flavored Markdown)**을 지원합니다.
* **굵게 (Bold)**: `**굵게**`
* *기울임 (Italic)*: `*기울임*`
* ~~취소선 (Strikethrough)~~: `~~취소선~~`

## 2. 코드 블록 및 강조
개발자를 위한 코드 하이라이팅이 아름답게 제공됩니다.

```javascript
// 자바스크립트 샘플 코드
const greeting = "Private Wiki에 오신 걸 환영합니다!";
console.log(greeting);
```
